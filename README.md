# 4-
ОПИСАНИЕ ПРОГРАММЫ

Программа "Система управления АТС" предназначена для учета клиентов телефонной станции, управления тарифами и регистрации звонков с автоматическим расчетом их стоимости. Программа позволяет вести базу данных клиентов, создавать различные тарифные планы, в том числе со скидками, и фиксировать все совершенные звонки с расчетом их стоимости.

Главное окно программы разделено на четыре вкладки: Клиенты, Тарифы, Звонки и Информация. На вкладке Клиенты отображается список всех зарегистрированных клиентов с их контактными данными. На вкладке Тарифы можно просмотреть все доступные тарифные планы с указанием стоимости минуты разговора и типа тарификации. Вкладка Звонки содержит полную историю всех совершенных звонков с указанием клиента, направления, длительности и стоимости. На вкладке Информация отображается общая статистика по системе - количество клиентов, тарифов и звонков.


РАБОТА С КЛИЕНТАМИ

Для добавления нового клиента необходимо перейти на вкладку Клиенты и нажать кнопку Добавить. Откроется окно для ввода данных клиента. В этом окне нужно заполнить три обязательных поля: Фамилия, Имя и Телефон. Фамилия и имя должны содержать минимум два символа и состоять только из букв. Допускается использование пробелов и дефисов, например для двойных фамилий. Номер телефона должен содержать от десяти до пятнадцати цифр. Можно вводить номер в любом удобном формате, например с круглыми скобками, пробелами и дефисами. После заполнения всех полей нужно нажать кнопку OK и новый клиент появится в таблице.

Для изменения данных существующего клиента нужно выбрать его в таблице, кликнув по соответствующей строке, и нажать кнопку Изменить. Откроется то же окно, но с уже заполненными данными клиента. При этом окно будет иметь желтый фон, увеличенный размер шрифта и заголовок Редактировать клиента вместо Добавить клиента. Эти визуальные отличия помогают понять, что вы находитесь в режиме редактирования, а не добавления нового клиента. После внесения изменений нужно нажать кнопку Сохранить.

Удаление клиента производится выбором его в таблице и нажатием кнопки Удалить. Программа попросит подтвердить удаление, так как эта операция необратима. Важно понимать, что при удалении клиента автоматически удаляются все его звонки из базы данных. Это сделано для обеспечения целостности данных - не может быть звонков без клиента.

На вкладке Клиенты также доступна кнопка Сортировать, которая упорядочивает список клиентов по фамилии в алфавитном порядке. Это удобно, когда в базе много клиентов и нужно быстро найти конкретного человека.


РАБОТА С ТАРИФАМИ

Добавление нового тарифа начинается с перехода на вкладку Тарифы и нажатия кнопки Добавить. В открывшемся окне нужно заполнить несколько полей. Поле Город должно содержать название города или направления, по которому действует данный тариф. Название должно состоять минимум из двух букв. В поле Цена за минуту указывается стоимость одной минуты разговора в рублях. Можно вводить как целые числа, так и дробные, используя точку или запятую в качестве разделителя. Цена должна быть больше нуля и не превышать один миллион рублей.

Далее необходимо выбрать тип тарифа из выпадающего списка. Доступны два варианта: Regular и Discounted. Тип Regular означает обычный тариф без скидок, при котором стоимость звонка рассчитывается простым умножением длительности на цену за минуту. Тип Discounted предполагает наличие скидки. Если выбран этот тип, становится активным поле Скидка, в которое нужно ввести процент скидки от нуля до ста. При расчете стоимости звонка сначала вычисляется полная стоимость, затем от нее вычитается указанный процент.

Например, если создан тариф на Москву стоимостью три с половиной рубля за минуту типа Regular, то десятиминутный звонок будет стоить тридцать пять рублей. Если же создать тариф на Санкт-Петербург стоимостью два рубля восемьдесят копеек за минуту типа Discounted со скидкой пятнадцать процентов, то десятиминутный звонок сначала рассчитается как двадцать восемь рублей, затем вычтется скидка четыре рубля двадцать копеек, и итоговая стоимость составит двадцать три рубля восемьдесят копеек.

Редактирование тарифа выполняется аналогично редактированию клиента - выбираем тариф в таблице, нажимаем Изменить, вносим изменения и сохраняем. Окно редактирования имеет зеленый фон для визуального отличия от режима добавления. При удалении тарифа также удаляются все звонки, совершенные по этому тарифу. Сортировка тарифов производится по названию города в алфавитном порядке.


СОВЕРШЕНИЕ ЗВОНКОВ

Для регистрации звонка нужно сначала выбрать клиента на вкладке Клиенты, затем нажать кнопку Совершить звонок. Если в базе еще нет ни одного тарифа, программа выдаст сообщение о том, что сначала нужно добавить тарифы. Если тарифы есть, откроется окно для оформления звонка.

В этом окне сначала нужно выбрать тариф из выпадающего списка. В списке отображаются все доступные тарифы с указанием города, цены за минуту и типа тарификации. Далее нужно ввести длительность звонка в минутах. Обратите внимание, что можно вводить только целые числа - дробные минуты не поддерживаются. По мере ввода данных в нижней части окна автоматически пересчитывается и отображается итоговая стоимость звонка с учетом выбранного тарифа и его скидки, если она предусмотрена.

После проверки всех данных нужно нажать кнопку Совершить. Звонок будет зарегистрирован в системе и отобразится на вкладке Звонки. Программа покажет сообщение с итоговой стоимостью звонка для подтверждения операции.


ПРОСМОТР ЗВОНКОВ И СТАТИСТИКИ

На вкладке Звонки отображаются все зарегистрированные в системе звонки. Каждая запись содержит идентификационный номер звонка, номера клиента и тарифа, направление звонка, его длительность в минутах и рассчитанную стоимость в рублях. Из этой таблицы можно удалить любой звонок, выбрав его и нажав кнопку Удалить звонок.

Для просмотра всех звонков конкретного клиента нужно выбрать любой его звонок в таблице и нажать кнопку Звонки клиента. Откроется окно со статистикой, в котором будут перечислены все звонки данного клиента с указанием направления, длительности и стоимости каждого звонка. В конце также отображается общая длительность всех звонков и их суммарная стоимость. Это удобно для анализа активности клиента и подсчета его общих расходов.

Вкладка Информация предоставляет общую статистику по системе. Здесь отображается три показателя: общее количество клиентов, общее количество тарифов и общее количество совершенных звонков. Эти данные автоматически обновляются при любых изменениях в базе.


СОХРАНЕНИЕ И ЗАГРУЗКА ДАННЫХ

В нижней части главного окна программы расположены кнопки для работы с файлами. Кнопка Сохранить в файл позволяет сохранить все данные программы в файл на жестком диске. При нажатии на эту кнопку откроется стандартное диалоговое окно выбора места сохранения. Рекомендуется сохранять файлы с расширением ats, хотя это текстовый файл и его можно открыть в любом текстовом редакторе. В файл сохраняются все клиенты, все тарифы и все звонки, то есть полное состояние базы данных на момент сохранения.

Кнопка Загрузить из файла позволяет восстановить данные из ранее сохраненного файла. При нажатии откроется диалоговое окно выбора файла. Важно понимать, что загрузка данных из файла полностью заменяет текущие данные в программе. Поэтому перед загрузкой программа попросит подтвердить это действие. Если в текущей сессии были внесены изменения, которые не сохранены в файл, они будут потеряны после загрузки данных из файла.

Кнопка Обновить перечитывает все данные из базы и обновляет отображение во всех таблицах. Это может быть полезно, если кажется, что данные отображаются некорректно. Кнопка Очистить всё полностью удаляет все данные из системы - всех клиентов, все тарифы и все звонки. Эта операция также требует подтверждения, так как является необратимой.

В самом низу окна программы отображается метка времени последнего обновления данных. Каждый раз, когда вы добавляете, изменяете или удаляете какие-либо данные, эта метка автоматически обновляется, показывая текущую дату и время. Это помогает отслеживать, когда были внесены последние изменения.
